# Backend Structure

Projeto backend em Node.js utilizando Express, Knex (PostgreSQL), Passport para autenticação e Swagger para documentação da API.

---

## Tecnologias

- Node.js
- Express
- Knex (PostgreSQL)
- Passport (JWT e autenticação)
- Swagger (Documentação)
- bcryptjs
- dotenv
- consign (auto load de módulos)
